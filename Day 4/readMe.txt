README

Project Structure:
This framework contains two packages: "page" and "test".

1. page
   This package contains the SeleniumHandlePage class.
   It includes reusable Selenium functions used across the questions.

2. test
   This package contains separate files for each question.
   Example: question1.py, question2.py, etc.

How to Run:
You have to run these files from the "Day 4" folder:

1. Open the "Day 4" folder in the IDE.
2. In the terminal, run the required test file using this command:

   python -m test.question1

For other questions, replace question1 with the required file name.
